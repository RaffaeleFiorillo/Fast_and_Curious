Esta pasta contém todos os textos do jogo. Estes texto são usados para o utilizador digitar durante as partidas.
- Cada ficheiro txt corresponde à um texto para o jogo;
- Cada texto deve ter um número máximo de carateres por linha (48);
- Cada texto deve ter um número máximo de linhas (6);
- Em todos os textos, a primeira linha corresponde ao número de linhas do respetivo texto;
- As imposições de limite foram estabelecidas para garantir a melhor visibilidade possivel durante as partidas;
